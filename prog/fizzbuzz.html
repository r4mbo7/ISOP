Fizz Buzz

Le Fizz Buzz est un jeux ou il suffit de compter, mais attention! Quand un nombre est multiple de 5 le joueur doit dirt "Buzz"(et pas 5), quand c'est un multiple de 7 le joueur doit dirt "Fizz" et quand c'est un multiple de 35 le joueur doit dirt "FizzBuzz"! C'est simple non? Mais essay de compter jusqu'a 50 ;).

Input X
Repeat X=N
N+1->N
If partDec(N/35)=0
Then
Disp "FIZZBUZZ"
ElseIf partDec(N/5)=0
Then
Disp "BUZZ"
ElseIf partDec(N/7)=0
Then
Disp "FIZZ"
Else
Disp N
END
